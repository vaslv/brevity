# Brevity — Документация

Сервис коротких ссылок с маршрутизацией по правилам, учётом кликов и
исходящими колбеками. Laravel 13 + Filament 5 + PostgreSQL, запускается
под Laravel Sail.

Эта папка — канонический справочник для людей и агентов, работающих над
проектом.

## Карта

- [GLOSSARY.md](./GLOSSARY.md) — двуязычная (EN ↔ RU) терминология.
  **Начинай отсюда, если не уверен, как что-то назвать.**
- [ARCHITECTURE.md](./ARCHITECTURE.md) — стек, модель данных,
  поток запроса, ключевые подсистемы.
- [ADMIN.md](./ADMIN.md) — админка Filament: ресурсы, навигация,
  соглашения по i18n.
- [DEVELOPMENT.md](./DEVELOPMENT.md) — запуск, тесты, релизы,
  соглашения.
- [../SDK_API.md](../SDK_API.md) — публичный HTTP API-контракт (лежит
  в корне репо, потому что его потребляют внешние авторы SDK).

## Аудитория

Написано для двух читателей:

1. Новый контрибьютор, который впервые видит репо.
2. AI-агент (Claude Code, Codex, Cursor, …), вызванный без
   предварительного контекста.

Оба должны уметь ответить на «как работает X?» не грепая кодбазу, и
понимать, где вносить изменения.

## Поддержание документации в актуальном виде

Эти файлы — часть проекта, а не внешние заметки. Обновляй их в том же
коммите, где меняется поведение или терминология. Если документация
разошлась с реальностью — это баг.
